# Mezzio Quick-Start

A static documentation site that teaches PHP developers how to get started with
[Mezzio](https://docs.mezzio.dev) — the PSR-15 middleware microframework — by
building their first JSON API. Read it top to bottom (~60 min) and you finish
with a working API.

## What it covers

18 sections: introduction and installation, project structure, the request
lifecycle, routing, middleware, piping and stacking, request handlers, the
service container, configuration, a step-by-step Products API worked example,
database setup with PDO and Doctrine ORM, third-party service integration
(mailer, Stripe), error handling, and testing.

Every core concept follows the same structure — **What / Why / How / Mental
model / Use case** — with the mental models styled as distinct callout boxes,
Mermaid diagrams for the request lifecycle and middleware stack, and working
PHP code samples throughout.


This guide is unofficial and unaffiliated with the Laminas Foundation. Code
samples target Mezzio 3.x on PHP 8.1+.
